# Kubernetes Power Utilities

A small CLI tool for helping with k8s cluster interactions.

## Installation

Simply run
```shell
go install github.com/Diaphteiros/kpu@latest
```
or clone the repository and run
```shell
task install
```

> [!NOTE]
> This project uses [task](https://taskfile.dev/) instead of `make`.
